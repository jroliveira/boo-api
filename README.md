# boo (api)

Boo (api) is an open source project written in Scala (Akka HTTP) that implements technique Feature Toggle.

### Installing

``` bash
$ git clone https://github.com/jroliveira/boo-api.git
```

### How to use it

``` bash
$ sbt
> run
```
